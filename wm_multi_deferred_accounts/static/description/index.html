Use specific Deferred Expense/Revenue Accounts in Odoo 18.0.

Note: This app works only if your "Generate Expense/Revenue Entries" setting set to on "Bill/Invoice Validation"

Features
  • Ability to use specific Deferred Account for each deferred Expense/Revenue created.
  • Ability to use configure products with deferred Expense/Revenue accounts.
